# tslcparts Coming Soon Banner

This project is a simple "Coming Soon" banner for the company **tslcparts**. It is designed to inform visitors that the website is under construction and will be available soon.

## Project Structure

```
tslcparts-coming-soon
├── public
│   └── index.html          # Main HTML document
├── src
│   ├── index.js           # Entry point for the JavaScript application
│   ├── components
│   │   └── ComingSoonBanner.js # Functional component for the banner
│   ├── styles
│   │   └── main.css       # CSS styles for the project
│   └── utils
│       └── colors.js      # Color scheme constants
├── package.json            # npm configuration file
├── .gitignore              # Files to ignore in version control
└── README.md               # Project documentation
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd tslcparts-coming-soon
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the "Coming Soon" banner.

## Customization

You can customize the banner by modifying the `ComingSoonBanner.js` component and the styles in `main.css`. The color scheme can be adjusted in the `colors.js` file.
